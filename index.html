麻醉前氣道評估 Pre-anesthesia airway assessment

整合多項評估工具，綜合判斷困難氣道風險：

  • LEMON：L=外觀、E=3-3-2法則、M=Mallampati、O=阻塞、N=頸部活動度
  • Mallampati I~IV：Class III/IV 與困難喉鏡視野相關
  • 其他警示：頸部放射治療史、阻塞性睡眠呼吸中止症、BMI >35、下頷退縮
  • 護理師角色：確認評估記錄已完成並於麻醉前簡報時提出；確認困難氣道車位置與設備完整性

考慮清醒插管 vs 全麻後插管 Awake intubation vs induction then intubation

優先考慮清醒插管的情境：

  • 多重困難氣道指標並存
  • 飽胃（嗜睡、腸阻塞）
  • 困難面罩通氣且困難插管
  • 病人無法耐受短暫缺氧
  • 口腔癌術後、頭頸部放射治療後
護理師準備清單
  • 纖維支氣管鏡（FOB）或視頻喉鏡備妥
  • 局部麻醉藥（Lidocaine spray/nebulization）
  • 鎮靜藥（Dexmedetomidine / 低劑量 Midazolam）依醫囑備妥
  • 必要時通知耳鼻喉科備術中支援

清醒插管流程 Awake intubation

標準步驟：

  • 充分表面麻醉：Lidocaine 噴霧 → 等待 3-5 分鐘
  • 輕度鎮靜（保留自主呼吸）
  • 選擇工具：纖維支氣管鏡（首選）/ 視頻喉鏡 / Awake direct laryngoscopy
  • 插管成功後確認位置（EtCO₂ + 胸部起伏）
  • 確認後才可給予肌肉鬆弛劑與誘導藥

啟動困難氣道應變流程，立即：

  • 呼叫支援（二線麻醉醫師或 ENT）
  • 面罩通氣維持氧合
  • 考慮是否喚醒病人
護理師立即行動
  • 啟動緊急呼叫系統
  • 推出困難氣道車
  • 記錄嘗試次數與時間
  • 準備 Sugammadex（逆轉 Rocuronium）

Plan A → B → C → D 決策流程

Plan A — 直接喉鏡 / 視頻喉鏡插管

◎
最大嘗試次數：3 次（含不同術者），超過即轉 Plan B。每次嘗試前重新評估可優化之處。
  • 體位優化：Ramped position（肥胖病人）/ 頸部伸展
  • BURP 手法 / 外部喉部壓迫（ELM）
  • 更換喉鏡片型號或換用視頻喉鏡
  • 使用 Stylet 或 Bougie 輔助
護理師角色
  • 執行 BURP / ELM（依醫師指示）
  • 持續監測 SpO₂，低於 94% 立即告知
  • 記錄每次嘗試工具、視野 Cormack-Lehane 分級

Plan B — 聲門上氣道裝置（SAD）

▲
Plan A 失敗後，首要目標為維持氧合，而非繼續嘗試插管。
  • 放置 Second-generation SAD（如 LMA Supreme、i-gel）
  • 確認通氣有效後評估：手術可否在 SAD 下進行？
  • 若需插管：可透過 SAD 使用 Aintree catheter + FOB 引導換管
  • 評估是否喚醒病人，擇期再處理
護理師角色
  • 立即備妥合適型號 LMA（依體重）
  • 通知術者目前氣道狀況，協調是否繼續手術
  • 若喚醒病人：準備拮抗藥物、準備解釋後續計畫給病人

Plan C — 面罩通氣 + 喚醒病人

▲
SAD 放置失敗時，嘗試面罩通氣。若可維持氧合，喚醒病人是最安全的選擇。
  • 雙人面罩通氣（兩手固定面罩 + 一人擠球）
  • 口咽 / 鼻咽氣道輔助器
  • 給予 Sugammadex 逆轉肌鬆
  • 病人清醒後重新評估，計畫清醒插管或其他策略
CICO — Cannot Intubate, Cannot Oxygenate
!
生命威脅情境。面罩通氣與 SAD 均失敗，SpO₂ 持續下降。立即進行緊急侵入性氣道建立。
  • 首選：環甲膜切開術（Cricothyrotomy）— Scalpel-Bougie-Tube 技術
  • Needle cricothyrotomy 為暫時過渡，不能替代 surgical airway
  • 若已有術前氣切計畫：立即執行緊急氣切
  • 全隊呼叫：ENT / 外科緊急支援
護理師緊急行動
  • 立即呼叫緊急代碼，通知外科 / ENT
  • 備妥環甲膜切開包 / 緊急氣切包
  • 推算 Succinylcholine / Ketamine 緊急用量
  • 記錄時間軸：SpO₂ 開始下降時間、每次介入時間
  • 體外心臟按壓準備（若心跳停止）

PPE 穿著標準（插管人員）

!
插管為產生氣溶膠程序（AGP），必須使用最高等級呼吸防護。
  • N95 口罩（或同等 FFP2/FFP3）+ 外科口罩覆蓋
  • 護目鏡 + 臉罩（Face shield）
  • 防護衣（Gown）+ 手套 × 2 層
  • 帽子 + 鞋套
▲
穿著 PPE 時手術靈活度下降，插管前應充分熟悉器械操作。建議指定最有經驗的麻醉醫師執行插管，減少嘗試次數。

人員最小化原則

插管時手術室內人員盡量減少，非必要人員在門外等待，插管完成並確認後再進入。

◎
決策：快速序列插管（RSI）+ 視頻喉鏡優先
  • 視頻喉鏡（如 GlideScope）優先：操作者與氣道保持較大距離，降低暴露
  • 避免 Bag-mask ventilation（BMV）：增加氣溶膠產生，使用 RSI 以避免
  • 插管前在病人臉上蓋布單（某些建議），插管後立即移除
  • 連接呼吸器前，先連接 HME filter（高效過濾器）
  • 氣管內抽吸使用密閉式抽吸管（Closed-suction system）
  • 手術期間維持 HEPA 過濾，手術室門保持關閉

插管前護理師任務

  • ✓確認負壓手術室已啟動（或最大通風設定）
  • ✓通知手術室門外等待人員，插管完成前不得入內
  • ✓備妥視頻喉鏡（GlideScope 已充電）
  • ✓備妥 HME filter，連接於呼吸器迴路
  • ✓確認自身 PPE 穿著正確（他人協助確認）
  • ✓密閉式抽吸管已連接
脫 PPE 注意（插管後）
  • 脫除順序：手套（外層）→ 護目鏡 → 防護衣 → 手套（內層）→ N95
  • 每一步驟後進行手部衛生
  • 脫除 N95 前確保已離開汙染區域

傳播途徑TB：Droplet nuclei（<5 μm），COVID-19：Droplet + Aerosol
懸浮時間TB 可達數小時；COVID-19 較短
呼吸防護兩者均需 N95 以上；TB 若有 PAPR 更佳
手術室兩者均建議負壓室；TB 結束後需更換空氣≥12 次
!
張力性氣胸為立即威脅生命情況。傳染控制不能延誤治療，需同步進行。
◎
緊急胸腔引流（無需全麻）→ 若病情允許，再評估後續手術需求

  • 續行胸腔引流管（第 5 肋間，腋中線）
  • 若需全麻插管：同 COVID-19 AGP 防護標準執行
  • 氣管可能因氣胸而偏移，插管時確認深度與雙側呼吸音
  • 避免使用 Nitrous oxide（加重氣胸）

傳染控制協調

知識題
執行 Cricoid pressure（Sellick's maneuver）時，以下哪一點描述正確？
正確答案：C

Cricoid pressure 的時機為：誘導藥注入開始施壓 → 確認 ETT 位置正確、氣囊充氣後解除。施壓點為環狀軟骨（不是甲狀軟骨），向後壓迫食道以減少胃內容物逆流。

注意：若施壓影響插管視野，可依麻醉醫師指示適度減壓或調整角度。過強的 Cricoid pressure 反而可能使喉部偏移，增加插管困難。
情境決策
Plan A 插管失敗（已嘗試 3 次），面罩通氣 SpO₂ 維持 96%。您準備執行 Plan B 放置 LMA。此時護理師最應優先確認的是？

知識題
為 COVID-19 確診病人進行插管，以下哪一項措施有助於降低操作者的感染風險？
正確答案：C — 視頻喉鏡

視頻喉鏡（如 GlideScope）讓操作者可在較大距離外操作，同時螢幕顯示聲門視野，減少臉部靠近氣道的需要，顯著降低氣溶膠暴露風險。

選項 A：直接喉鏡需操作者臉部靠近，增加暴露。選項 B：BMV 是高度產生氣溶膠的操作，在 AGP 情境下應盡量避免，採 RSI 以省略此步驟。選項 D：空氣傳染病人手術室應使用負壓（負壓室），防止污染空氣外溢，而非正壓。
情境決策
口腔癌術後病人（放射治療後，張口度 1.5cm），計畫清醒鼻插管。局部麻醉完成後，您發現病人突然焦慮、SpO₂ 下降至 90%。最可能的原因及下一步？
正確答案：B — 鎮靜過深，給予支持並喚醒

清醒插管的核心原則是「維持自主呼吸」。若鎮靜藥（如 Dexmedetomidine 或 Midazolam）劑量過大，造成呼吸抑制，最安全的處置是給予氧氣支持、暫停鎮靜藥、等待病人恢復自主呼吸後再繼續。

此時不應立即 RSI（口腔癌張口 1.5cm，全麻後面罩通氣和 SAD 均困難，RSI 失敗風險極高）。選項 D 在 SpO₂ 下降時強行插管，可能因病人不合作而失敗。
知識題
活動性肺結核（塗片陽性）手術結束後，手術室至少需要換氣多少次（Air changes per hour，ACH）且多長時間，才能讓下一台手術使用？
正確答案：B — ACH ≥ 12，等待 ≥ 60 分鐘

WHO 與 CDC 建議，活動性 TB 病人離開後，空間需至少 12 次 ACH，且等待時間依通風量計算（一般標準建議 ≥60 分鐘）才能確保空氣中結核菌核滴核濃度降至安全水準。

消毒噴霧和紫外線可作為輔助，但不能替代充分的換氣時間。紫外線對懸浮在空氣中的飛沫核效果有限，且需要足夠的照射時間與覆蓋範圍。

高風險拔管流程（DAS 指引）
▲
以下任一情況適用高風險流程：術中插管困難、面罩通氣困難、氣道水腫、長時間俯臥位手術、頭頸部手術後。
1
術前規劃：「拔管計畫書」
手術開始前即確認拔管策略，不在術後臨時決定
  • 與術者討論：手術是否影響氣道解剖（頸部血腫風險、水腫預期）
  • 決定拔管時機：手術室內 vs PACU vs ICU
  • 決定是否使用 Airway Exchange Catheter（AEC）
  • 確認備用設備就位：視頻喉鏡、SAD、再插管所需物品
2
AEC 放置（高風險病人）
拔管前預先置入 AEC，保留再插管引導通路
  • AEC 為中空導管，可保留在氣管內供再插管引導用，也可接氧氣供氧
  • 置入深度：成人 23–25 cm（門齒），勿過深（隆突損傷風險）
  • 拔管時固定 AEC，抽氣囊，移除 ETT，AEC 暫時留置
  • 病人清醒、氧合穩定 30–60 分鐘後再移除 AEC
  • 若需再插管：AEC 引導下直接套入新 ETT
3
拔管前準備
確認所有標準達標，設備備妥後執行
  • 100% O₂ 預充氧 3–5 分鐘
  • 口腔及氣管內分泌物完全吸引乾淨
  • 確認再插管設備在手可及處（視頻喉鏡、ETT、SAD）
  • 血壓、心率在可接受範圍（拔管刺激可引發高血壓/心跳加速）
  • 必要時給予 Lidocaine 1 mg/kg 減少拔管嗆咳反射
4
執行拔管
清醒拔管（首選）或深麻醉拔管（特定適應症）
  • 清醒拔管（優先）：病人清醒、保護性反射恢復後拔管，高風險病人的標準選擇
  • 深麻醉拔管（特定）：適用咳嗽禁忌（眼科、顱內壓、血管吻合）；氣道必須確定不困難
  • 拔管前最後口腔抽吸，抽癟氣囊，病人吸氣末或深呼吸時拔出 ETT
  • 立即換上面罩供氧，觀察呼吸型態
5
拔管後 30 分鐘監測
高風險期，護理師持續在旁
  • SpO₂ 持續監測，目標 >95%
  • 觀察呼吸型態：有無喘鳴（stridor）、吸氣性困難
  • 頸部手術：觀察頸部腫脹、血腫徵象
  • 若出現喉痙攣或呼吸窘迫 → 啟動對應處置流程
  • AEC 尚未移除者：確認 AEC 固定、病人未自行拔除

辨識

  • 拔管後突發性高調喘鳴或完全無呼吸音（完全性喉痙攣）
  • SpO₂ 迅速下降，呼吸費力，胸骨上窩凹陷
  • 常見誘因：拔管時刺激、分泌物、淺麻醉

處置

  • 立即 100% O₂ 正壓面罩通氣，CPAP 25–40 cmH₂O
  • Larson's maneuver：雙手食指強力按壓下頷骨與乳突間凹陷
  • 若無改善：Succinylcholine 0.1–0.5 mg/kg IV（小劑量打破痙攣）
  • 嚴重：再插管 + 全劑量 Succinylcholine 1.5 mg/kg
  • 記錄時間，觀察是否出現負壓性肺水腫
護理師行動
  • 立即呼叫麻醉醫師
  • 備妥面罩 + 氧氣，執行正壓通氣
  • 抽取 Succinylcholine 依醫囑備用
高風險 負壓性肺水腫 NPPO +

機轉

  • 喉痙攣後病人用力吸氣 → 胸腔內負壓急遽升高 → 肺微血管液體外滲
  • 通常在喉痙攣解除後 30–60 分鐘內出現

辨識

  • 拔管後出現粉紅色泡沫狀分泌物
  • SpO₂ 下降，呼吸急促，肺部囉音
  • 胸部 X 光：雙側肺浸潤

處置

  • 高流量 O₂，必要時 CPAP / BiPAP
  • 嚴重：再插管，PEEP 通氣
  • 利尿劑（Furosemide）可考慮
  • 通知 ICU 準備，持續監測
中度風險 殘餘肌鬆效果 +

辨識

  • 抬頭無力、握力差、呼吸淺快
  • SpO₂ 緩慢下降，病人焦慮感
  • TOF ratio < 0.9

處置

  • Rocuronium 逆轉：Sugammadex 2 mg/kg（TOF ≥2 個顫搐）或 16 mg/kg（無顫搐）
  • Succinylcholine 無需逆轉（短效，自然代謝）
  • 傳統逆轉：Neostigmine 0.04 mg/kg + Atropine 0.02 mg/kg
  • 給予逆轉後待 TOF > 0.9 再拔管
中度風險 術後喉嚨痛 / 聲音沙啞 +

原因

  • ETT 對聲帶及氣管黏膜的機械性刺激
  • 氣囊壓力過高（>30 cmH₂O）造成缺血
  • 手術時間長、頭頸部手術、多次插管嘗試

評估

  • 術後詢問病人：喉嚨痛程度（NRS）、聲音變化
  • 嚴重聲音沙啞或持續 >48 小時 → 考慮 ENT 評估（聲帶麻痺？杓狀軟骨脫位？）

處置

  • 輕度：冷飲、安撫，通常 24–48 小時內自然改善
  • Lidocaine 潤喉糖或含漱（若醫院有）
  • 記錄於麻醉後訪視紀錄
